# Product Hunt copy

**Name:** VibeGuard
**Tagline (60 chars max):** Security guardrails for AI-generated web apps

**Description:**
VibeGuard catches the 21 most common security flaws in AI-generated ("vibe-coded") web
apps — exposed secrets, broken access control, missing RLS, injection — before they ship.
It works three ways from one OWASP-mapped catalogue: a Claude/Cursor rules pack for
secure-by-default generation, CI gates that block the unsafe stuff, and hardened Next.js +
Supabase starters. A `governance.yaml` manifest keeps it proportional and lightweight.
Open source (MIT), research-backed, honest about scope.
